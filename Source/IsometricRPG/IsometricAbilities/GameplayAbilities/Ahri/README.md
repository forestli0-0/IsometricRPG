# 阿狸 (Ahri) 技能组实现

## 概述
这个文件夹包含了基于Gameplay Ability System (GAS) 的英雄联盟阿狸技能组的完整实现。所有技能都遵循LOL的原版设计，包括伤害数值、冷却时间和机制。

## 技能列表

### Q技能 - 欺诈宝珠 (Orb of Deception)
**文件**: `GA_Ahri_Q_OrbOfDeception.h/cpp`  
**类型**: 方向性技能 (SkillShot)  
**冷却时间**: 7秒  
**魔法消耗**: 65点  

**描述**:
- 阿狸投掷出一个魔法宝珠，向目标方向飞行并穿透敌人造成魔法伤害
- 宝珠在达到最大距离后会返回阿狸身边，返回途中再次对敌人造成魔法伤害
- 同一个敌人可以被宝珠的去程和返程分别命中一次

**技术要点**:
- 继承自 `GA_SkillShotProjectileAbility`
- 实现返回机制和重复命中检测
- 基础伤害: 40 + 40*技能等级 + 0.35*AP

### W技能 - 狐火 (Foxfire)
**文件**: `GA_Ahri_W_Foxfire.h/cpp`  
**类型**: 自我施放 (SelfCast)  
**冷却时间**: 9秒  
**魔法消耗**: 55点  

**描述**:
- 阿狸释放三个狐火球环绕在自己周围
- 每个狐火球会自动寻找附近的敌人并造成魔法伤害
- 狐火球优先攻击阿狸当前攻击的目标，如果没有目标则寻找最近的敌人
- 同一个敌人只能被狐火球命中一次

**技术要点**:
- 继承自 `GA_SelfCastAbility`
- 实现自动目标寻找和环绕特效
- 基础伤害: 40 + 25*技能等级 + 0.3*AP（每个狐火球）

### E技能 - 魅惑 (Charm)
**文件**: `GA_Ahri_E_Charm.h/cpp`  
**类型**: 目标指向投射物 (Targeted Projectile)  
**冷却时间**: 12秒  
**魔法消耗**: 85点  

**描述**:
- 阿狸吹出一个吻，向目标方向发射一个爱心投射物
- 命中的第一个敌人会被魅惑，被迫朝阿狸移动一段时间，并受到魔法伤害
- 被魅惑的敌人移动速度会逐渐减慢，期间受到的伤害增加20%

**技术要点**:
- 继承自 `GA_TargetedProjectileAbility`
- 实现魅惑状态效果和伤害增幅
- 基础伤害: 60 + 35*技能等级 + 0.5*AP

### R技能 - 狐狸大招 (Spirit Rush)
**文件**: `GA_Ahri_R_SpiritRush.h/cpp`  
**类型**: 自我施放 + 位移 (SelfCast + Dash)  
**冷却时间**: 100秒  
**魔法消耗**: 100点  
**使用次数**: 3次（10秒内）  

**描述**:
- 阿狸向鼠标指针方向瞬移一段距离，并发射三道狐灵火球攻击附近的敌人
- 该技能可以在10秒内连续使用3次，每次使用都会重置冷却时间
- 狐灵火球会优先攻击阿狸当前的目标，然后是附近的敌人
- 同一个敌人被多个狐灵火球命中时，后续命中的伤害会递减

**技术要点**:
- 继承自 `GA_SelfCastAbility`
- 实现多次使用机制和位移系统
- 基础伤害: 60 + 45*技能等级 + 0.35*AP（每个狐灵火球）

## 技能特色机制

### 1. 魅惑连招加成
当阿狸的E技能魅惑敌人后，其他技能对被魅惑目标的伤害会增加20%。

### 2. 技能组合
- **Q+E连招**: 先E魅惑敌人，然后Q技能可以保证命中并享受伤害加成
- **W+R连招**: 使用R技能接近敌人后，W技能的狐火球能更容易找到目标
- **满连**: E魅惑 → Q宝珠 → W狐火 → R突进 → 持续输出

### 3. 位移安全性
阿狸的R技能提供了很强的机动性，可以用于：
- 进攻：接近敌人进行爆发输出
- 逃跑：连续3次位移脱离危险
- 追击：追击残血敌人

## 开发注意事项

### 依赖关系
- 需要投射物基类 `AProjectileBase`
- 需要伤害系统的 `GameplayEffect` 类
- 需要位移任务 `AbilityTask`
- 需要Niagara特效系统

### 扩展性
每个技能都留有足够的扩展接口：
- 特效和音效可以在蓝图中配置
- 伤害数值可以通过属性调整
- 状态效果可以通过GameplayEffect系统扩展

### 网络同步
所有技能都考虑了网络同步：
- 投射物使用Replicated
- 位移使用客户端预测
- 状态效果通过GAS自动同步

## 使用说明

1. 确保所有依赖的基类和系统已正确设置
2. 在角色的AbilitySystemComponent中注册这些技能
3. 为每个技能配置相应的输入绑定
4. 在蓝图中设置特效、音效和动画资源
5. 调整技能的数值平衡
